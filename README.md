# dice_bot

[BCDice](https://github.com/bcdice/BCDice)を使ったダイスボット。

## 使い方

`/roll`でダイスを振ることができます。

## セルフホスト

1. リポジトリをクローンします。
2. `bundle install`します。
3. Discord Bot PortalからBotを作成します。
4. `.env.sample`を`.env`にコピーし、トークンを入力します。
5. `discorb setup`を実行します。
6. `discorb run`を実行します。
